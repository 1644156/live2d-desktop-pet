# 打字机效果设计文档

## 一、功能概述

| 特性 | 描述 |
|------|------|
| 自适应速度 | 根据消息长度自动调整（短消息慢，长消息快） |
| 智能停顿 | 句号/逗号/颜文字自动增加停顿 |
| 语义气口 | 长句主谓宾断点自动加100ms停顿，模拟换气感 |
| 情绪卡顿 | 害羞/紧张/表白场景自动插入卡顿 |
| 思考前置 | 长句先显示"唔..."等思考词，再开始打字 |
| 输入打断 | 用户开始输入时自动跳过并隐藏气泡 |
| 向上扩展 | 气泡随内容增加向上扩展 |

## 二、核心参数

```javascript
// 自适应速度
const BASE_SPEED = 80;  // 基础速度
const MIN_SPEED = 30;   // 最小速度
const MAX_SPEED = 100;  // 最大速度

// 标点停顿规则
const PAUSE_MARKS = {
    '。': 300, '！': 250, '？': 250,
    '，': 150, '、': 100, '～': 100, '…': 200
};

// 语义断点关键词
const SEMANTIC_BREAKS = [
    '其实', '那个', '就是说', '怎么说呢', '然后', '所以',
    '但是', '不过', '而且', '因为', '虽然', '如果',
    '我觉得', '我认为', '我想', '你知道吗'
];

// 情绪卡顿配置
const EMOTION_STUTTER = {
    '我喜欢': { position: 'before', delay: 500, endDelay: 800 },
    '我爱你': { position: 'before', delay: 600, endDelay: 1000 },
    '那个...': { position: 'after', delay: 400 },
    '我...': { position: 'after', delay: 500 },
    '其实我': { position: 'after', delay: 300 },
    '一直想': { position: 'after', delay: 200 },
};

// 思考前置词库
const THINKING_PREFIXES = [
    { text: '唔...', minLength: 20, delay: 1000 },
    { text: '呃...', minLength: 30, delay: 1200 },
    { text: '那个...', minLength: 15, delay: 800 },
    { text: '让我想想...', minLength: 40, delay: 1500 },
    { text: '嗯...', minLength: 10, delay: 800 },
];
```

## 三、完整打字流程

```
showMessage(text, emotion) 被调用
         ↓
┌─────────────────────────────────────┐
│ 1. 判断是否需要思考前置              │
│    - 长句(>20字) → 显示"唔..."      │
│    - 停顿 1000-1500ms               │
│    - 清除思考词                      │
└─────────────────────────────────────┘
         ↓
┌─────────────────────────────────────┐
│ 2. 开始逐字打字                      │
│    for each character:              │
│    - 计算自适应速度                  │
│    - 检查标点停顿                    │
│    - 检查语义气口                    │
│    - 检查情绪卡顿                    │
│    - 显示字符                        │
│    - 动态调整气泡大小                │
└─────────────────────────────────────┘
         ↓
┌─────────────────────────────────────┐
│ 3. 打字完成                          │
│    - 触发自动隐藏计时器              │
└─────────────────────────────────────┘
```

## 四、文件修改清单

| 文件 | 修改内容 |

| `public/bubble.html` | 添加 `typeWriter()` 函数、停顿逻辑、打断处理 |
| `src/app/bubble_window.py` | 添加 `skip_typing()` 方法、WebChannel信号 |
| `src/app/input_window.py` | 添加输入开始时通知气泡打断 |

## 五、打断机制

用户开始输入 → InputWindow检测焦点 → 通知BubbleWindow → 调用JS的skipTyping() → 立即显示完整内容 → 隐藏气泡

## 六、参数汇总表

| 参数类型 | 参数名 | 值 | 说明 |
|---------|--------|-----|------|
| 基础速度 | BASE_SPEED | 80ms | 基础打字间隔 |
| 标点停顿 | PAUSE_MARKS | 100-300ms | 句号/逗号等 |
| 语义气口 | SEMANTIC_BREAK | 100ms | 换气感停顿 |
| 情绪卡顿 | EMOTION_STUTTER | 300-1000ms | 害羞/紧张 |
| 思考前置 | THINKING_DELAY | 800-1500ms | 开口前停顿 |
